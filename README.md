# Aprendizaje Automático de Máquina I (2026-02)

Este repositorio contiene los notebooks para el curso introductorio de Machine Learning (Aprendizaje Automático de Máquina I), ofrecido para la maestría MACC en la Universidad del Rosario (Bogotá, Colombia) durante el periodo 2026-02.

**Notebooks del Curso de Aprendizaje Automático:**

* **ML_Session1:** Introducción al aprendizaje automático, tipos de problemas, análisis exploratorio de datos, esquema general de Machine Learning para clasificación e introducción a k-vecinos más cercanos (KNN).
* **ML_Session2:** Métricas de evaluación (accuracy, precisión, recall, f1-score), matriz de confusión, optimización de hiperparámetros, búsqueda en grilla y validación cruzada.
* **ML_Session3:** Regresión logística, curva ROC y ajuste de umbral.
* **ML_Session4:** Árboles de decisión, poda (pruning) y parámetro de complejidad (ccp).
* **ML_Session5:** Problemas de regresión, regresión lineal simple y múltiple, mínimos cuadrados y métricas de evaluación (Error Cuadrático Medio - MSE).
* **ML_Session6:** Otros modelos de regresión, incluyendo KNN-regressor y Decision tree regressor.
* **ML_Session7:** Modelos de Clustering (aprendizaje no supervisado), K-means y DBSCAN.
* **ML_Session8:** Introducción a Redes Neuronales, neurona, perceptrón, funciones de activación, idea de Backpropagation y redes en Keras+Tensorflow.
